Mach-Zehnder Interferometry and Erasure of 'Which-path' Information

(Experiment Code: 2.21)

Description

One of the most astounding and bewildering depictions of quantum mechanics is the single- particle interference experiment. If a source that emits only one particle (such as photons or electrons) at a time, is used in a classic double slit experiment, an interference patterns will emerge. This pattern is built up over multiple passes of identical particles. Interestingly, the pattern could be switched on and off even after light has emerged from the interferometer.

The Mach-Zehnder Interferometer

The Mach-Zehnder interferometer is an amplitude splitting interferometer which divides the incoming beam into two paths. The path differences result in an interference pattern. This contrasts with a waveform splitting device such as the classic Young’s double slit interferometer. We are going to use a Mach-Zehnder interferometer to verify the basic principles of polarization, obtainment and eventually the erasure of ‘which-path’ information. This so called ‘erasure’ experiment underpins deep questions about quantum mechanics as we will highlight later. The first step of course is setting up and aligning the basic interferometer (see next section). The interferometer’s layout is shown in Figure 1.

Interference and Erasure of Which Path Information

If we set up the Mach-Zehnder interferometer above with a single photon source whose emission rate is so low that there exists only one photon in the whole setup at one point in time, we still observe an interference pattern that builds up over time. This pattern is due to the wave nature of quantum particles. Now if we try to measure which path the photon has actually taken, the interference pattern disappears. This is because in quantum mechanics, measuring a system leads to irreversible changes to the system.

Obtaining ‘which-path’ information and collapse of fringes

In this part of the experiment, we place P2 and P3 in the paths I and II respectively. The mutual angle between the polarizers P2 and P3 is adjustable. The schematic diagram with two more polarizers is shown in Figure 2.

If the polarizers P2 and P3 are crossed, ‘which-path’ information is obtained, and the interference pattern collapses while if P2 and P3 are parallel, ‘which-path’ information is absent and interference fringes are observable. These scenarios are depicted in Figure 3.

Conclusion

A single photon can show interference pattern. The question arises that which path it takes to get to the detector. When we try to observe its path by adding polarizer 2 and 3, the wave function collapses and we see no interference pattern as if photon has lost its wave nature and become a particle.
